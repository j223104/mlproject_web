# 데이터 종류 및 레이블링 방식#
1. 데이터:   바이낸스 api 를 통해 비트코인/달러 직전 21년 11월 1일~ 22년 1월 13일 데이터
2. 이미지 만들기:  한 이미지 : 5분 봉 20개(100분) matplotlib로 차트 생성하여 이미지 저장
3. 레이블링 : 
수익률 =  현재 데이터(직전 100분데이터)의 마감 가격 / 15분 뒤의 마감가격
수익률 >=0.7% : 매수 시그널 (0)
0<수익률<0.7% : 유지  (1)
수익률 <0  :  매도 시그널 (2)
